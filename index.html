The World Wide Web

The Web is made up of HTML documents, HTML stands for Hyper Text Mark Up Language and is the basis for virtually every web page. HTML documents are made up of Text Content (what you see), Markup (what it looks like), reference to other docs and links. Hyperlinks connects pages together. The major components of the web are:

1) The user/browser

2) The internet

3) Servers

Tags

Tags are pretty interesting, and once understood not intimidating at all, at least so far. Tags essentially tell our computer what to do with the contents within it. There is typically an opening and closing tag.
There is no need to worry about remembering all tags, there countless resources available online that can be used to research tags. I plan to use them!
